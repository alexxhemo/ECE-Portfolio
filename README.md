# Engineering Portfolio — Alexander Xhemo

This repository contains a collection of my engineering and software projects across hardware design, embedded systems, VLSI, C/C++/Python development, Linux systems programming, and full-stack web development.

The goal of this portfolio is to show the range of projects I have worked on, including circuit design, PCB layout, microcontroller programming, digital/VLSI design, backend/frontend development, and algorithmic software projects.

---

## Portfolio Areas

### PCB Design

This section includes PCB design projects involving schematic capture, component selection, board layout, routing, and manufacturing outputs.

Projects include:

- Optical transceiver PCB
- Audio amplifier PCB
- Schematic and layout documentation
- Gerber/manufacturing files where applicable
- Design screenshots and project notes

Tools used:

- Altium Designer
- KiCad
- Circuit simulation tools
- PCB layout and routing workflows

Large generated simulation files, such as `.raw` files, are intentionally excluded from the repository to keep the project clean and lightweight.

### VLSI Layouts and Schematics

This section contains VLSI design work, including transistor-level schematics, layouts, and timing analysis.

Projects include:

- Static master-slave register
- Full adder schematic and layout
- Sequential arithmetic circuit integration
- DRC and LVS verification work
- Setup time, hold time, and clock-to-Q delay analysis

Tools and concepts used:

- Cadence Virtuoso
- CMOS transistor-level design
- Layout design with shared diffusion
- DRC/LVS debugging
- Timing measurements and waveform analysis

### CPU Scheduler for Ubuntu

This project implements a CPU scheduling calculator and simulator for Linux/Ubuntu.

Implemented scheduling algorithms include:

- First-Come, First-Served
- Shortest Job First
- Round Robin
- Priority Scheduling

The project calculates and displays scheduling metrics such as:

- Waiting time
- Turnaround time
- Completion time
- Average performance metrics

Tools and concepts used:

- C/C++
- Ubuntu/Linux environment
- Process scheduling concepts
- Command-line compilation and testing
